Década de los 90

Regresar a Linea de tiempo evolución de la web

Era de la WEB 1.0

1990 WEB 1.0

Web 1.0 se refiere a un estado de la World Wide Web. Personas conectándose a la Web y la Web como punto de información estática.

1990 World Wide Web.

Sistema de distribución de documentos de hipertexto o hipermedia interconectados y accesibles a través de Internet. Con un navegador web, un usuario visualiza sitios web compuestos de páginas web que pueden contener textos, imágenes, vídeos u otros contenidos multimedia.

1994 Creación de Amazon.

Compañía estadounidense de comercio electrónico y servicios de computación en la nube a todos los niveles con sede en la ciudad estadounidense de Seattle.

1996 Lanzamiento de CSS.

Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

1998 Creación de Google.

Es una compañía cuya especialización son los productos y servicios relacionados con Internet, software, dispositivos electrónicos y otras tecnologías.

1999 Lanzamiento de XML.

Es un metalenguaje que permite definir lenguajes de marcas desarrollado por el World Wide Web Consortium utilizado para almacenar datos en forma legible.
